# Swiggy-Sales-Analysis-End-to-End-Data-Analytics-Project
This project performs a complete end-to-end data analysis on Swiggy food delivery transactions across multiple states, cities, restaurants, categories, and dishes. The goal is to extract meaningful insights from raw transactional data by applying data cleaning, dimensional modelling (Star Schema), and KPI-driven business analytics.


## Dataset Description

The raw dataset swiggy_data includes the following fields:

| Column Name     | Description                      |
| --------------- | -------------------------------- |
| State           | State from where order is placed |
| City            | City of the customer             |
| Order_Date      | Order placed date                |
| Restaurant_Name | Vendor name                      |
| Location        | Locality of customer             |
| Category        | Cuisine / food category          |
| Dish_Name       | Name of dish ordered             |
| Price_INR       | Price of the dish                |
| Rating          | Rating given by customer         |
| Rating_Count    | Number of ratings received       |
